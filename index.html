Comment fonctionne le traitement par Lot ?

Ce mode de traitement massif ne modifie pas les images dans votre navigateur (ce serait trop lent). Il fonctionne comme une télécommande :

  1. Réglez la Date/Heure et/ou le GPS ci-dessous.
  2. Cliquez sur "Générer le script (.bat)".
  3. Double-cliquez sur le script pour l'exécuter. Votre PC va utiliser le moteur ExifTool (installé en coulisses avec cette application) pour traiter le dossier sélectionné instantanément !

Ces actions se basent sur la "Date de prise de vue" interne à chaque photo. Les fichiers sans date ne seront pas traités.

⚠️ Attention : Assurez-vous d'avoir suffisamment d'espace disque disponible sur votre support cible (Clé USB, Disque) avant de dupliquer un très gros lot d'images.

Outils de lecture et de nettoyage

Générez un fichier pour extraire dans un tableau CSV (lisible par Excel / Google Sheets) le nom, modèle, dimensions, date et GPS des photos. Cela n'affectera pas vos actions suivantes.

Attention, outil puissant : générez un script qui va effacer totalement toutes les métadonnées (Dates, GPS, Appareil, Logiciel) pour un anonymat complet. Rassurez-vous, de nouvelles copies _geophoto.jpg seront créées car la protection des originaux est obligatoire sur cet outil.
⚠️ Espace Disque Requis : Ce processus va dupliquer intégralement vos fichiers. Prévoyez suffisamment d'espace libre sur votre support cible (Clé USB, Disque) avant de lancer ce script sur un gros dossier.

🛡️ Sécurité Windows : Lors du lancement d'un fichier .bat, Windows peut bloquer l'exécution ("Vos paramètres de sécurité Internet ont empêché l'ouverture..." ou un écran bleu).
Pour le débloquer : Faites un clic-droit sur le fichier téléchargé > Propriétés > Cochez la case "Débloquer" en bas > Appliquez.

Manuel d'Utilisation

Comment utiliser GeoPhoto Editor 📍📷

GeoPhoto Editor V3 propose trois méthodes de travail distinctes selon vos besoins : le Mode Simple (unitaire), la Sélection Visuelle (gestion par tableau) et le Traitement par Lot (moteur industriel ExifTool).

📸 A. MODE SIMPLE (Image par image - JPEG uniquement)

1. Principe

  • Glissez une photo JPEG. Si elle a déjà un GPS, la carte se positionne dessus.
  • Modifiez la position d'un clic sur la carte ou via la recherche.
  • Mettre à jour & Télécharger injecte les nouvelles coordonnées directement dans le fichier.

🖼️ B. SÉLECTION VISUELLE (Tableau de bord - JPEG uniquement)

🚀 Puissance industrielle : Ce mode utilise également ExifTool pour traiter vos photos sélectionnées instantanément via un script BATCH.

Comment ça marche ?

  • Étape 1 : Copiez/collez le chemin de votre dossier Windows (ex: C:\Photos\Vacances).
  • Étape 2 : Cliquez sur "Parcourir" et sélectionnez ce même dossier pour charger les miniatures.
  • Interaction : Cochez les photos à modifier dans le tableau. Cliquez sur une ligne pour voir l'aperçu et son GPS actuel.
  • Action : Réglez le GPS ou la Date dans le panneau de droite, puis cliquez sur "Appliquer (En mémoire)". Une étiquette verte s'ajoute devant le nom dans le tableau.
  • Sortie & Action : Cliquez sur "Générer Script Modif (.bat)". Déplacez ensuite ce fichier dans le dossier de vos photos et lancez-le pour valider les changements.

📁 C. TRAITEMENT PAR LOT (Massif - Tous formats RAW, JPG, PNG...)

🚀 Puissance industrielle : Ce mode utilise ExifTool pour traiter des centaines de fichiers (y compris les formats photo professionnels RAW) instantanément via un script BATCH.

Les 3 Onglets :

  • Métadonnées : Force une date ou un GPS identique sur TOUTES les photos du dossier.
  • Organisation : Renomme les fichiers par date (20261231_1.jpg) et les range automatiquement dans des dossiers par Année/Mois.
  • Utilitaires : Exportez un listing Excel (CSV) de vos photos ou utilisez le "Kärcher" pour tout effacer (Anonymat).

Zéro transfert de données : Qu'il s'agisse du Mode Simple ou du Traitement par Lot, vos fichiers originaux ne sont jamais envoyés sur nos serveurs. L'entièreté des modifications est traitée localement par votre propre hardware.
